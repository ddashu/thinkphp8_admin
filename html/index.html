为什么选择量子Admin

从权限控制到内容管理，从 AI 集成到支付对接，一站式解决企业后台需求

RBAC 权限模型

用户 → 角色 → 菜单（按钮）三级权限体系，支持细粒度到按钮级别的访问控制，灵活适配各类组织架构。

🔐

JWT 安全认证

Access Token + RefreshToken 双令牌机制，支持无感续期，保障接口安全的同时提供流畅的登录体验。

📊

数据可视化看板

基于 ECharts 构建的数据看板，实时展示用户趋势、API 调用分布、模型使用情况，让运营数据一目了然。

📋

多渠道支付配置

内置微信支付、支付宝双渠道配置管理，支持沙箱测试、回调地址配置，一键测试连通性。

📔

富文本文章管理

集成 Quill 富文本编辑器，支持图文混排、分类管理、置顶推荐、发布下架，满足内容运营全流程。

📡

短信渠道管理

支持阿里云、腾讯云等多短信渠道并行管理，含每日用量统计、默认渠道切换，稳定可靠。

用户权限管理

完整的用户 CRUD 操作，支持状态切换、密码重置、搜索筛选。配合角色管理实现灵活的人员权限分配。

CRUD 操作 状态切换 搜索筛选

角色与权限授权

可视化树形菜单勾选授权，支持目录/菜单/按钮三级权限粒度。一个角色可关联多个菜单，一个用户可拥有多个角色。

树形权限 可视化勾选 三级粒度

AI API 密钥管理

集中管理各 AI 服务商的 API Key，支持密钥创建、编辑、启用/禁用操作，为上层 AI 应用提供统一密钥调度入口。

统一管理 状态控制 安全存储

富文本文章系统

集成 Quill 富文本编辑器，支持加粗/斜体/标题/列表/对齐/颜色/链接/图片等丰富格式。配合分类、封面、置顶、推荐等功能。

Quill 编辑器 分类管理 置顶推荐

多渠道支付配置

Tab 切换式配置界面，同时支持微信支付和支付宝。涵盖 AppID、商户号、私钥公钥、签名方式、回调地址等完整参数。

微信支付 支付宝 沙箱测试

短信渠道配置

表格化管理多个短信服务商渠道，每个渠道可独立配置 AccessKey、签名、模板，支持设置每日发送上限与默认渠道切换。

多渠道并行 用量监控 默认切换
